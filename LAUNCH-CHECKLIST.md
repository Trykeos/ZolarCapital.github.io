# ZolarCapital.com Launch Checklist

## Domain and email

- [ ] Confirm zolarcapital.com auto-renew is enabled.
- [ ] Enable two-factor authentication at the domain registrar.
- [ ] Add zolarcapital.com to Microsoft 365.
- [ ] Create primary mailbox, such as michael@zolarcapital.com.
- [ ] Create shared aliases, such as contact@zolarcapital.com and info@zolarcapital.com.
- [ ] Configure SPF.
- [ ] Configure DKIM.
- [ ] Configure DMARC.

## Website

- [ ] Choose host: Cloudflare Pages, Netlify, Vercel, Webflow, or Framer.
- [ ] Upload or rebuild the static site.
- [ ] Connect zolarcapital.com.
- [ ] Redirect www.zolarcapital.com to zolarcapital.com or vice versa.
- [ ] Test desktop pages.
- [ ] Test mobile pages.
- [ ] Test navigation.
- [ ] Test contact form.
- [ ] Review privacy page with counsel.

## CRM and campaign manager

- [ ] Create lead webhook endpoint.
- [ ] Map website form fields to CRM contact fields.
- [ ] Create standard tags.
- [ ] Confirm consent capture.
- [ ] Trigger internal lead notification.
- [ ] Add unsubscribe handling for marketing emails.
- [ ] Test a real inquiry.

## Analytics

- [ ] Create Google Analytics property.
- [ ] Add tracking script.
- [ ] Verify Google Search Console.
- [ ] Submit sitemap.
- [ ] Check page titles and descriptions.
- [ ] Confirm Open Graph image displays correctly in link previews.
- [ ] Replace placeholder mockups with product screenshots when available.

## Post-launch

- [ ] Send the website to trusted contacts for feedback.
- [ ] Add first three insight articles.
- [ ] Replace mockups with product screenshots as tools mature.
- [ ] Add case studies when available.
